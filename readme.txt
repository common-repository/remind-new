== Description ==
If you modify a post, this plugin will put a icon beside the post for the next three days to call attention to the visitors and when the visitors hover their mouse on that icon, some string will show detailing the modified date
 
== Installation ==
1. Upload the folder `remind-new` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
